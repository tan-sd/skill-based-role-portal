Font Awesome Icons

To add icons using Font Awesome, go to https://fontawesome.com/icons and search for the icon you want

Remember to click the Free button (paid ones don't work)

Copy the HMTL code snippet and paste in HTML document: (eg. '<i class="fa-solid fa-circle-user"></i>')

Remember to insert the following Font Awesome import JS tag at the bottom of the HTML body:
<script src="https://kit.fontawesome.com/c7ddd7a650.js" crossorigin="anonymous"></script>
